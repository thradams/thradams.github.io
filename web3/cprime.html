C23 front end compiler

** This is a preview only **

This is a C23 front end compiler written from scratch by (twitter @thradams) as a hobby project.

  1. preprocessor updated C23
  2. syntax analisys
  3. some microsoft extensions
  4. The semantic analysis (incomplete)
  5. Outputs C code. It can read C23 and ouputs C99 for instance
  6. preserve (if possible) code formation and macros

ouput

CPrime was compiled with emscripten and it is running in your browser.

It can be used for...

  1. Build and visit C23 AST
  2. Syntax analisys
  3. Experiments
  4. As a preprocessor?

Previous version http://thradams.com/web2/cprime.html

This is the version 3 and the code is 100% new.

Version 2, builds the source from the AST adding macros, comments etc...

Version 3, just changes the tokens. For instance, if we write code with attributes and we want to compile do C11, we need to remove these tokens, otherwise they will be there inside the ouput. (This feature is missing now, I need to visit each node from the ast...)

This version already compiles itself
